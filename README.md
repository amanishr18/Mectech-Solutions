# MecTech Solutions Website

This repository contains the source code for the official MecTech Solutions static website.

## Project Structure

- `index.html`: The main HTML document.
- `css/style.css`: The stylesheet extracted for clean separation of concerns.
- `js/script.js`: The JavaScript logic for page interactions like the mobile menu and scroll animations.

## Deployment

This is a static website designed to be easily hosted directly via [GitHub Pages](https://pages.github.com/). Assuming standard settings, any pushes to the main branch will deploy to the live site.

To run it locally, you can simply open `index.html` in any web browser, or use a local development server like Live Server in VS Code.
